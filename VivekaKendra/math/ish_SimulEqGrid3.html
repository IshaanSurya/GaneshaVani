Simultaneous Equation - 3
  Represent items in each row in the form of an equation and calculate the value of the each item.


Example:
 Equations:
 3a + 4b = 10  ---> (Eq 1)
 9a + 2b = -10  --->(Eq 2)

 Equation 2 multiplier :   -2  (If no multiplier needed then enter    None   in the input box)
 Equation after multiplication :    -18a - 4b = 20   [ -2( 9a+2b=-10)  => -18a-4b=20  ]
 Term that cancels out :    4b , -4b 
 Remaining (Residual) equation :   3a -18a = 10+20   
 Simplified residual equation :  -15a = 30  
  Residual variable value (a) :   a=-2   [  a = 30/-15 => -2  ]
 Substituting the value in Eq :   -6+4b=10  [  substituting a=-2 in Eq1  ]
 Equation for the other variable :   4b=16  [  4b=10+6   ]
 Value of the other variable (b) :   b=4    (4b = 16 => b = 16/4)
