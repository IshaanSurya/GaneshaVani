Pythagoras Theorem -6

 
Observe the right angle triangles given below and determine the length of the hypotenuse .




Example: 
Apply Pythagoras Theorem.
a2 + (4a)2  = (√272)2   =>  a2 + 16a2  = 272
=>  17a2  = 272    =>  a2  = 272 / 17
=>  a2  = 16    =>  a = 4 (ignore -4 ; length cannot be negative )
Expected Answer.
 272  =  a2  +  16a2  =  17a2    
  a2  =  272  /  17  =  16
